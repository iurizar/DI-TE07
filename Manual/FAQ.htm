FAQ - Preguntas frecuentes

  1. ¿Cómo accedo a la información de cada comercial?

    El dashboard te la opción de seleccionar entre diferentes comerciales que trabajan en la empresa. Una vez seleccionado el comercial mediante el Menú de Comerciales podremos acceder a su información detallada mediante el Menú de consulta de datos. Para más información sobre qué tipo de datos puedes consultar, haz click aquí.

  2. ¿Por qué no puedo acceder al menú de consulta de datos?

    Para poder consultar los datos de cada comercial primero tienes que elegirlo. El menú de consulta de datos está desactivado por defecto, puesto que sin comercial no tiene información que mostrar. Una vez seleccionado el comercial mediante el Menú de comerciales la barra de consulta se activará y podrás acceder a la información.

  3. ¿Cómo puedo consultar las ventas mensuales de un comercial?

    El dashboard permite consultar esta información de diferentes maneras. Se puede acceder a una lista detallada de las ventas mensuales desde la ventana de Facturación total haciendo click en el botón de Detalle. Para una comparativa en forma de gráfico, el apartado Resumen de Ventas mostrará un gráfico comparativo entre las ventas a diferentes empresas.
